Magnify app screenshots for the case-study gallery.

Save your 4 screenshots into THIS folder with these EXACT names
(all lowercase, .png):

  overview.png      -> the "Overview" screen (gauges, Run Scan)
  observatory.png   -> the "Observatory" screen (Environment Health 100)
  privacy.png       -> the "Privacy" screen (Privacy Risk 60)
  priorities.png    -> the "Priorities" screen (findings graph)

PNG or JPG both work; if you use JPG, change the extensions in
projects.js (the magnify.gallery list) to .jpg.

Once the files are here, the gallery on case.html?p=magnify shows them
automatically in a sharp framed layout. Until then it shows a clean
"preview coming" placeholder — never a broken image.
